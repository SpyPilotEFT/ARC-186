# ARC-186

SPT 4.0.13 mod that adds the AN/ARC-186 R/T as a custom electronics barter item.

Includes the Integrated Avionics Mechanic questline and an ARC-186 hideout craft. If the craft does not appear immediately after completing Integrated Avionics - Part 2, clear the SPT user cache and restart the server/client.

## Install

1. Download the latest release zip.
2. Extract `Spys_ARC-186` into `SPT/user/mods`.
3. Start SPT.
